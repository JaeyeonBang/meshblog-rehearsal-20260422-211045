Paper, ink, and a quiet kind of depth.

흑백만 쓰고 깊이는 오직 빛과 그림자로 표현한다. 정보 위계는 종이의 튀어나옴과 눌림 — 떠오른 것은 집을 수 있고, 눌린 것은 상태를 품는다. 이 한 규칙이 모든 컴포넌트를 관통한다.

00

원칙

왜 이렇게 생겼는가
흑백 only
색상 대비로 강조하지 않는다. 모든 강세는 명도 + 음영 + 타이포그래피 weight로만 표현한다. 채도는 warm-trace (hue 60, chroma 0.003) 수준으로만 허용.
표면 하나, 그림자 둘
바탕과 카드가 같은 색이다. 카드는 빛(위)과 그림자(아래)로만 분리된다. 경계선은 거의 쓰지 않는다 — 음영이 곧 구조다.
Raised vs Inset
집을 수 있는 것(버튼, 카드, 뱃지)은 밖으로, 받아들이는 것(입력, 선택됨, 코드)은 안으로. 같은 요소여도 상태에 따라 방향이 뒤집힌다.
Serif reads, sans navigates
읽는 글은 Literata. UI 라벨, 버튼, 메타는 Pretendard. Mono는 라우트, 토큰 이름, 코드에만.
토큰으로만 그린다
모든 값이 :root의 CSS 변수. 컴포넌트 내부에 하드코딩된 색/그림자/radius 금지. 테마 교체 시 한 블록만 바뀐다.
고요함이 기본값
애니메이션은 240ms ease-out-quint 한 종류. 바운스 없음, 스프링 없음. 종이가 움직이는 것처럼.

음영 · Elevation

뉴모피즘의 심장

그림자 한 쌍 (밝은 빛 좌상 + 어두운 그림자 우하)이 모든 입체감을 만든다. 여섯 가지 조합으로 요소의 성격과 상태를 표현한다.

raised · sm기본 카드, 뱃지, 버튼 idle — 살짝 떠 있음
raised강조 카드, 버튼 hover — 집을 수 있음
raised · lg모달, 플로팅 패널 — 확실히 위

05

Button

primary · secondary · ghost · icon
Primary
페이지당 하나. 어두운 잉크색 위에 sans weight 500.
Secondary
기본 표면 위에 떠 있음. 여러 개 병렬 가능.
Ghost · Icon
음영 없이 놓여 있다가 hover 시에만 떠오른다.
Pressed state
active 시 inset으로 뒤집힘. 상태 지속형(토글) 버튼도 동일 공식.
Disabled
그림자 제거 + opacity 0.4. 눌리지 않는다는 걸 음영으로 알린다.
Button group
같은 축의 선택지. 하나만 inset으로 눌림.

sunk 표면 · 흑백 신택스
block
문맥에서 한 단계 가라앉은 표면. 신택스는 weight만으로 구분.
# retrieve.py
def retrieve(q, k=5):
    vectors = index.search(q, k)
    return [v.id for v in vectors]
inline

Theming · JSON 주입

우하단 버튼으로 열어서 실시간 변환

모든 디자인 토큰은 단일 JSON 객체로 추출되어 있습니다. 사용자가 자신의 디자인 가이드를 이 형식으로 제공하면, 키 하나당 CSS 변수 하나가 교체되어 전체 사이트가 재스타일링됩니다. 구조는 건드리지 않고 값만 바꿔요.

우측 예시는 이 페이지와 blog.html 양쪽 모두에 적용됩니다. 우하단의 테마 버튼을 누르면 JSON 편집기가 열리고, 프리셋도 3개 준비돼 있어요. 저장해두면 다음 방문 시에도 유지됩니다 (localStorage.meshblog_theme). 

// 현재 적용된 테마
const theme = {
  "hue": 60,
  "surface": "oklch(.930 .003 60)",
  "surface-elev": "oklch(.955 .003 60)",
  "surface-sunk": "oklch(.905 .003 60)",
  "ink": "oklch(.220 .005 60)",
  "ink-2": "oklch(.360 .005 60)",
  "ink-3": "oklch(.520 .005 60)",
  "f-serif": "Literata, ...",
  "f-sans": "Pretendard, ...",
  "r-md": "14px",
  "r-pill": "999px"
};
